# Scientific Calculator (C++)

## Overview
A simple console-based Scientific Calculator built using C++. It performs basic arithmetic and common mathematical operations through a menu-driven interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Square Root
- Power
- Trigonometric Functions (if implemented)

## Technologies Used
- C++
- Standard Template Library (STL)
- cmath library

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/scientific-calculator.git
   ```
2. Compile the program:
   ```bash
   g++ calculator.cpp -o calculator
   ```
3. Run the executable:
   ```bash
   ./calculator
   ```
